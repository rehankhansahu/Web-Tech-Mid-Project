🎯 **SkinConsult Pro - Smart Skincare Consultation Website**
SkinConsult Pro is a responsive web-based skincare consultation platform that helps users identify their skin type, explore skincare articles, book appointments, and get personalized recommendations.

---

🚀 **Features**
🧴 Skin Type Selection (Oily, Dry, Combination, Sensitive)
📖 Skincare Articles with Modern Card Layout
📅 Appointment Booking (Stored on Screen / Local Storage)
💡 Product Recommendations Based on Skin Type
❓ Interactive FAQ Section (Expand/Collapse)
📞 Contact Form with Validation
📱 Fully Responsive Design (Mobile First)
🎨 Modern UI using Bootstrap & CSS3

📌 **How It Works**
Open the website in browser
Explore the homepage and features
Select your skin type to get details and recommendations
Read skincare articles
Book an appointment using the form
View submitted appointment on the same page
Explore FAQs and contact section
